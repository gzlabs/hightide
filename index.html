Hightide

A fast, reliable and scalable Java web framework.

Hightide makes writing web applications in Java joyful again!

Being tired of all the verbosity and boilerplate code/configuration a classic Java web application comes with. Inspired from all this ideas that new lightweight frameworks from non-Java ecosystems brought in. Based on the philosophy of RoR, adapted to the strength and stability of the JVM. Hightide is here to fill this gap.

Creating a new application is as simple as typing:

$ hightide new my-awesome-java-app

Build around the latest development trends and JVM technology.

Based on one of the most advanced VMs on the market (Java SDK 1.8) and one of the fastest web servers (Undertow). Hightide is set to become a new standard in Java web development.

Install Hightide

For mac users using Homebrew.

$ brew install hightide
